## Generating Your Own Reports

```bash
# HTML dashboard
bun --bun tools/readiness-check.ts --format=html --output=my-report.html

# JSON (for CI/automation)
bun --bun tools/readiness-check.ts --format=json > my-report.json

# Markdown (for docs)
bun --bun tools/readiness-check.ts --format=md > my-report.md
```
